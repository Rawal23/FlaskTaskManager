# Flask Task Manager

A simple task management web application built with Flask and SQLite.

## Features

- Create, read, update, and delete tasks
- Track task status and priority
- Responsive UI using Bootstrap

## Requirements

- Python 3.10+
- Flask
- Flask-SQLAlchemy
